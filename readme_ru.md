# VidSubX

![версия Python](https://img.shields.io/badge/Python-3.12-blue)
![поддерживаемые ОС](https://img.shields.io/badge/OS-Windows-green.svg)

Бесплатная программа для извлечения жёстко вшитых субтитров из видео и создания внешнего файла субтитров.

<img src="docs/images/gui%20screenshot.png" width="500">

**Возможности**

- Автоматическое обнаружение области субтитров путём поиска типичной зоны.
- Ручное изменение размера или положения области субтитров (кликните и перетащите мышью).
- Одиночное и пакетное обнаружение и извлечение субтитров.
- Возможность выбрать начальную и конечную позиции для извлечения субтитров (используйте клавиши со стрелками для точной настройки).
- Изменение масштаба отображения видео (Увеличить (Ctrl+Плюс), Уменьшить (Ctrl+Минус)).
- Можно скрыть часть видео без субтитров, чтобы избежать спойлеров.
- **Поддержка нескольких GPU**: CUDA (NVIDIA), DirectML (AMD), OpenVINO (Intel)
- **Автоматическое определение**: Система определяет доступные GPU-провайдеры.
- **Выбор в интерфейсе**: Пользователь может выбрать GPU-провайдер в настройках.
- **Мониторинг производительности**: Отдельная оптимизация для CPU и GPU.
- **Логирование информации о GPU**: Подробные сведения о видеокартах.
- **Резервная совместимость**: Автоматический переход на CPU, если GPU недоступен.
- Автоматическая оптимизация конфигурации для CPU для наилучшей производительности.
- Скомпилированные версии можно запускать без установленного Python.
- Всплывающие уведомления (Toast Notification) в Windows по завершении обнаружения и извлечения субтитров.
- [Документация по настройкам](docs/Preferences.md) для изменения параметров при извлечении субтитров.
- Поддержка множества языков через [PaddleOCR](https://github.com/PaddlePaddle/PaddleOCR). Они будут автоматически загружаться по мере необходимости.

**[Поддерживаемые языки и их сокращения](docs/Supported_Languages.md)**

### Полученные субтитры можно перевести с помощью этого [скрипта](https://github.com/voun7/Subtitle_Translator) или [SubtitleEdit](https://github.com/SubtitleEdit/subtitleedit).

## Инструкции по установке

### Скачивание и установка:

[Последняя версия Microsoft Visual C++ Redistributable](https://learn.microsoft.com/ru-RU/cpp/windows/latest-supported-vc-redist)

**Установка пакетов**

Для GPU

```
pip install -r requirements-gpu.txt
```

Для CPU

```
pip install -r requirements-cpu.txt
```

**_Примечание:_** `onnxruntime` и `onnxruntime-gpu` не должны быть установлены одновременно.

Запустите `gui.py` для использования графического интерфейса и `main.py` для использования терминала.

### Инструкции по компиляции

Запустите `compiler.py` для сборки программы.

Если антивирус выдаёт предупреждение, файл можно отправить [сюда](https://www.microsoft.com/ru-ru/wdsi/filesubmission) для анализа.
